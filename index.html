Fab(ricate)

Fabricate is a declarative modular project management and build tool, similar to maven but more domain specific and more concise. Below is a minimal scala library project configuration example.

application:
  name: &name test
  org: co.myco.proj
  version: 0.8.1
  author: My Name
  description: &desc An App
dependencies:
  - {conf: compile, org: joda-time, name: joda-time, rev: "2.0" }
  - {conf: compile, org: commons-lang, name: commons-lang, rev: "2.6", transitive: false}
